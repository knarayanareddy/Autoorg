The Philosophical Pillars

AutoOrg is built on the synthesis of the three most powerful multi-agent paradigms.

The Karpathy Constraint

Instructions, constraints, and stopping criteria carried simultaneously in one plain-Markdown file. Agents operate within a strictly defined sandbox of your design.

The Ratchet Principle

A relentless keep-or-revert scoring loop. Every single agent output must clear the judge's gate. Only validated improvements are committed to the Git audit trail.

MiroFish Grounding

No hallucinated drift. Agents operate within a dynamically updated knowledge graph reflecting actual entities, relationships, and pressure points.

The KAIROS Memory System

A three-tier memory architecture (Always-Loaded Index, On-Demand Fact Files, Search-Only Transcripts) designed for infinite scaling without context window fatigue.

Swarm Economics

Phase 10 integration with internal wallets and credit-based task delegation. Org-to-Org settlements for autonomous specialist subcontracting.

Model Agnostic & Local-First

Native support for Ollama, Anthropic, OpenAI, and Gemini. Run your entire research organization locally for maximum privacy and zero latency.

Specialist Personas

An organization is only as strong as its specialists. Each with a unique cognitive bias.

CEO

The Orchestrator. Coordinates task assignments via the flat-file mailbox. Integrates agent outputs into higher-order syntheses.

Engineer

The Producer. Directly drafts section contents, code, and structured data. Highly creative but grounded by Critic feedback.

Critic

The Glass-Half-Empty. Use "Steel Man then Destroy" methodology. Finds the single most fatal flaw in every proposal.

DevilsAdvocate

The Divergent Thinker. Argues against consensus. Prevents premature closure and corporate groupthink.

Archivist

The Memory. Prevents reinventing the wheel. Queries failed experiments to ensure the team learns from every revert.

RatchetJudge

The Authority. Scores against the immutable constitution.md. Decision is final. Uses highest-tier models only.

DreamAgent

The Synthesizer. Operates during the autoDream phase to resolve contradictions and distill ephemeral observations into absolute facts.
